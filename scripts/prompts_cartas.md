# Prompts para Generar Arte de Cartas - UG DECK

## Especificaciones Técnicas del Área de Arte

| Propiedad | Valor |
|-----------|-------|
| Área de arte en la plantilla | **200 x 78 píxeles** |
| Posición en la plantilla 400x400 | x: 100→300, y: 122→200 |
| Proporción | **2.56:1** (apaisado/landscape) |
| Resolución recomendada para generar | **400 x 156 px** (2x para mejor calidad) |
| Formato | PNG con fondo transparente o de color sólido |

> [!IMPORTANT]
> Las imágenes son **muy anchas y bajas** (tipo banner). Los personajes/objetos deben estar centrados y NO ocupar toda la altura, dejando un pequeño margen.

---

## Prompt Base (Template)

Copiá este prompt y reemplazá `{DESCRIPCION_VISUAL}` con la descripción de cada carta:

```
Genera una imagen en estilo pixel art retro de 16-bit para un juego de cartas universitario.

ESPECIFICACIONES TÉCNICAS:
- Resolución: 400 x 156 píxeles (proporción 2.56:1, muy ancho y bajo, tipo banner)
- Estilo: pixel art limpio, colores vibrantes, contornos definidos de 1-2px
- Fondo: transparente o color sólido plano (sin gradientes)
- Sin texto, sin letras, sin números, sin UI
- El sujeto principal debe estar centrado y ocupar ~70-80% del espacio
- Paleta de colores limitada (máximo 16-24 colores)

TEMÁTICA: Vida universitaria argentina, humor estudiantil, estética de videojuego retro

ESCENA: {DESCRIPCION_VISUAL}
```
